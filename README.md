## 📌 Deskripsi
Repository ini berisi kumpulan tugas praktikum Java yang bertujuan untuk memahami konsep dasar Pemrograman Berorientasi Objek (Object Oriented Programming / OOP).

Materi yang dipelajari meliputi:

- Constructor
- Parameter
- Enkapsulasi (Encapsulation)
- Inheritance (Pewarisan)
- Polymorphism
- Study Case
